
# Food Delivery Pricing & Rating Analysis

This project presents an end-to-end Excel-based analysis of food delivery data,
focusing on pricing patterns, rating reliability, and geographic demand distribution.

## Tools Used
- Microsoft Excel
- Pivot Tables & Charts

## Project Highlights
- Data cleaning and feature engineering within Excel
- Monthly, weekly, and quarterly trend analysis
- Dashboard creation with KPIs
- Documented insights and limitations

A dashboard preview is available in the `dashboard/` folder.
